# Phase 03: UI Components (Clone Web)
Status: ⬜ Pending

## Objective
Xây dựng tập hợp các components UI cơ bản tái sử dụng giống thiết kế của website nội bộ.

## Requirements
### Functional
- [ ] Tuân thủ Library có sẵn (tuỳ chọn NativeWind, hoặc thư viện Component chuẩn nhóm).
- [ ] Chạy mượt mà trên cả iOS lẫn Android.

### Non-Functional
- [ ] Giữ được style Guide như web (Màu sắc chính, Font chữ, Layout Input/Button).

## Implementation Steps
1. [ ] Cấu hình Theme / Stylesheet chung.
2. [ ] Viết các Base Component: Button, Input, Icon, Header, Card, Modal.
3. [ ] Viết List/Grid layout cho danh sách đơn hàng/sản phẩm.

## Files to Create/Modify
- `src/components/common/*`
- `src/theme/*`

## Test Criteria
- [ ] So sánh trực tiếp App Component và Web Component xem đạt bao nhiêu % độ tương đồng (mục tiêu >90%).
